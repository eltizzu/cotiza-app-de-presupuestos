Manual de Uso — Cotiza

Presupuestos de reformas y trabajos técnicos a partir de tus propios datos de precios y rendimientos.

📅 Última actualización: 21 de mayo 2026 🔧 Para autónomos y pymes

¿Qué es Cotiza?

Cotiza es una herramienta para crear presupuestos de reformas, oficios y trabajos técnicos a partir de datos controlados por el usuario.

Precios + Rendimientos + Trabajos tipo = Presupuesto base editable

Cotiza no inventa precios ni decide por ti. Calcula una base usando tu información y te deja revisar, editar y cerrar el presupuesto.

Estructura de la app

La demo tiene cinco secciones principales que conviene usar en orden la primera vez:

1. Inicio

  • Datos del negocio
  • Moneda e impuesto
  • Backup local

2. Precios

  • Base de costes
  • Material, mano obra...

3. Rendimientos

  • Reglas de estimación
  • Horas/m², litros/m²...

4. Trabajos tipo

  • Plantillas reutilizables
  • Combinan precios+rendimientos

5. Presupuesto

  • Documento editable
  • PDF para el cliente

1. Inicio

En "Inicio" se cargan los datos generales del negocio que aparecerán en los presupuestos.

  • Nombre del negocio — ej: Reformas Norte
  • Moneda — ej: EUR
  • Impuesto por defecto — ej: 21%
  • Margen por defecto — ej: 20% (no se muestra al cliente)
El margen se usa para calcular el precio antes de impuestos. La impresión para el cliente no muestra el margen interno.

Backup local

Desde "Inicio" también puedes exportar e importar un backup en formato JSON con todos los datos de Cotiza.

⚠️ La demo guarda datos en el navegador. Si se borra el almacenamiento, se pueden perder datos. Exporta backup si cargas datos importantes.

3. Rendimientos

Los rendimientos son reglas para estimar cantidades o tiempos: cuántas horas lleva una tarea, cuánto material se consume por m², etc.

Ejemplo: Si "Horas pintura por m²" vale 0,18 y el trabajo tiene 25 m², Cotiza estima: 25 × 0,18 = 4,5 horas.

Variables disponibles: m² · cantidad · metros lineales

4. Trabajos tipo

Un trabajo tipo es una plantilla reutilizable que combina precios y rendimientos. Ejemplos: Pintar habitación, Cambiar grifo, Colocar suelo laminado.

Crear un trabajo tipo

  1. Ir a Trabajos tipo
  2. Escribir nombre y descripción breve
  3. Agregar partidas (cada una necesita: precio, tipo de cálculo, rendimiento/variable y multiplicador)
  4. Guardar el trabajo tipo

Tipos de cálculo de partidas

Usar rendimiento

Cuando ya existe una regla cargada en Rendimientos.

Ej: Precio → Hora oficial / Rendimiento → Horas pintura por m²

Cantidad directa

Cuando la partida depende directamente de m², cantidad o metros lineales.

Ej: Suelo laminado × m² × 1,08 (merma)

Cada trabajo tipo se puede duplicar para crear variantes sin empezar de cero. Ej: Pintar habitación → Pintar piso completo → Pintar local.

5. Presupuesto

En "Presupuesto" se crea el documento editable para el cliente.

Datos del presupuesto

  • Número (se propone automáticamente)
  • Cliente y dirección/obra
  • Validez y fecha
  • Estado: Borrador · Enviado · Aceptado · Rechazado

Calcular base

  1. Elegir un trabajo tipo
  2. Cargar m², cantidad o metros lineales según corresponda
  3. Pulsar Calcular base
Cotiza genera partidas usando los precios y rendimientos cargados. El resultado es una base, no un presupuesto cerrado automáticamente.

Editar y ajustar partidas

Después de calcular, cada línea se puede editar, reordenar, quitar, o ajustar en cantidad y precio unitario. También se pueden agregar líneas manuales extra (retirada de escombros, protección, reparación previa, etc.).

6. Guardar y seguir presupuestos

El botón "Guardar presupuesto" guarda en el historial local. Desde "Presupuestos guardados" puedes abrir, cambiar estado, eliminar o buscar por número, cliente o dirección.

El resumen comercial muestra: borradores, enviados, aceptados y rechazados con sus totales.

El historial es local de la demo. No es una base de datos en la nube.

7. Imprimir / Guardar PDF

El botón "Imprimir / Guardar PDF" abre la impresión del navegador. El documento para el cliente incluye: datos del negocio, datos del cliente, fecha, partidas, subtotal, impuesto y total. No incluye coste ni margen interno.

Revisa siempre el presupuesto antes de imprimirlo: cliente, número, fecha, validez, partidas, cantidades, precios y total.

Tutorial: primer presupuesto de pintura

Ejemplo completo para entender el flujo de principio a fin.

Paso 1 — Configuración

Ir a "Inicio": cargar nombre, EUR, impuesto 21%, margen 20%. Guardar configuración.

Paso 2 — Verificar precios

Ir a "Precios": asegurarse de tener Pintura plástica interior, Material auxiliar pintura, Hora oficial y Desplazamiento urbano.

Paso 3 — Verificar rendimientos

Ir a "Rendimientos": comprobar que existen Litros pintura por m², Horas pintura por m² y Desplazamiento por trabajo.

Paso 4 — Calcular base

  1. Ir a "Presupuesto"
  2. Elegir trabajo tipo: Pintar habitación
  3. Superficie m²: 25
  4. Pulsar Calcular base

Paso 5 — Ajustar

Revisar cantidades, precio unitario y totales. Agregar línea extra si hace falta (ej: Reparación pequeña de pared, 1 unidad, 35€).

Paso 6 — Guardar

Cargar cliente, dirección/obra, fecha, validez y estado Borrador. Pulsar "Guardar presupuesto".

Pasos 7 y 8 — Estado y PDF

Cuando se envía al cliente: cambiar a Enviado. Si acepta: Aceptado. Luego imprimir o guardar PDF.

Buenas prácticas

  • Revisar siempre el presupuesto antes de enviarlo
  • Mantener los precios actualizados
  • Ajustar los rendimientos según experiencia real
  • Usar trabajos tipo como base, no como verdad absoluta
  • Exportar backup si cargas datos importantes
  • No usar la demo como sistema fiscal, contable o legal

Límites de la demo

  • Los datos se guardan en el navegador (no en la nube)
  • No hay usuarios ni contraseñas
  • No hay facturación ni contabilidad
  • No hay control de obra ni importador de Excel todavía
La demo sirve para validar el flujo central de Cotiza: cargar tus datos, calcular una base editable y generar un presupuesto ordenado.
