Manual de Uso — Cotiza

Presupuestos de reformas y trabajos tecnicos a partir de tus propios precios, consumos y plantillas.

Actualizado: junio 2026 Para autonomos y pymes

Que es Cotiza

Cotiza es una herramienta para crear presupuestos de reformas, oficios y trabajos tecnicos a partir de datos controlados por el usuario.

Precios + Consumos y tiempos + Plantillas = Presupuesto base editable

Cotiza no inventa precios ni decide por el usuario. Calcula una base usando la informacion cargada y deja que la persona revise, edite y cierre el presupuesto.

Estructura de la app

La app tiene nueve secciones. Conviene usarlas en orden la primera vez.

1. Inicio

Datos del negocio, margen, impuesto y backup local.

2. Precios

Materiales, mano de obra, transporte y otros costes.

3. Consumos

Reglas de calculo: horas por m2, litros por m2, etc.

4. Plantillas

Trabajos tipo reutilizables.

5. Presupuesto

Documento editable, calculo y edicion de partidas.

6. Guardar

Historial, estados y busqueda.

7. Dashboard

Metricas de volumen y estados.

8. Clientes

Base de clientes desde presupuestos.

9. PDF

Impresion limpia para el cliente.

1. Inicio

En "Inicio" se cargan los datos generales del negocio: nombre, telefono, email, direccion, moneda, impuesto por defecto y margen por defecto.

El margen se usa para calcular el precio antes de impuestos a partir del coste estimado. La impresion para el cliente no muestra el margen interno.

Backup local

Desde "Inicio" tambien se puede exportar e importar un backup. El backup es un archivo JSON con todos los datos de Cotiza.

  • Si se inicia sesion, Cotiza guarda los datos en la nube automaticamente.
  • Sin sesion, la app funciona en modo local con localStorage.
  • Si se borra el almacenamiento del navegador, se pueden perder los datos locales.
  • Conviene exportar backup si se cargan datos importantes.
  • Importar un backup reemplaza los datos locales actuales.

3. Consumos y tiempos

En "Consumos y tiempos" se cargan reglas para estimar cantidades o tiempos. Un consumo responde preguntas como: cuantas horas lleva una tarea, cuanto material se consume por m2 o cuantas unidades se necesitan segun cantidad.

Campos: nombre, variable, resultado por unidad y unidad resultante.

Variables disponibles: m2, cantidad y metros lineales.
NombreVariableResultado por unidadUnidad

Ejemplo: si "Horas pintura por m2" vale 0,18 y el trabajo tiene 25 m2, Cotiza estima 25 × 0,18 = 4,5 horas.

4. Plantillas de trabajos

Una plantilla de trabajo es una receta reutilizable. Ejemplos: pintar habitacion, cambiar grifo, colocar suelo laminado, cambiar foco.

Crear una plantilla

  1. Escribir el nombre del trabajo y una descripcion breve.
  2. Agregar partidas. Cada partida necesita: precio asociado, tipo de calculo y multiplicador.
  3. Guardar la plantilla.

Tipos de calculo

Usar rendimiento

Sirve cuando ya existe una regla cargada en "Consumos y tiempos". Ejemplo: precio = Hora oficial, rendimiento = Horas pintura por m2.

Cantidad directa

Sirve cuando la partida depende directamente de m2, cantidad o metros lineales. Ejemplo: precio = Suelo laminado AC5, variable = m2, multiplicador = 1,08 (8% por merma).

Editar o duplicar

Cada plantilla se puede editar o duplicar. Duplicar sirve para crear variantes sin empezar de cero. Consejo: para empezar mas facil, duplicar una plantilla parecida y cambiarla.

5. Presupuesto

En "Presupuesto" se crea el documento editable. Campos: numero, cliente, direccion/obra, validez, fecha y estado.

Estados disponibles: Borrador, Enviado, Aceptado, Rechazado.

Crear presupuesto base

  1. Elegir una plantilla de trabajo.
  2. Cargar superficie m2, cantidad o metros lineales segun corresponda.
  3. Pulsar "Crear presupuesto base".

Cotiza genera las partidas y muestra una explicacion de cada calculo. Por ejemplo: 25 m2 × 0,18 horas/m2 = 4,5 horas → 4,5 × 28 EUR = 126 EUR. El usuario siempre puede editar el resultado.

Editar partidas

Despues de calcular, las lineas se pueden editar, reordenar, quitar o ajustar en cantidad y precio unitario. El resultado calculado es una base, no un presupuesto cerrado automaticamente.

Agregar lineas extra

Se pueden agregar conceptos manuales que no esten en la plantilla: retirada de escombros, proteccion especial, reparacion previa, desplazamiento adicional, ajuste manual.

6. Guardar y seguir presupuestos

El boton "Guardar presupuesto" guarda el presupuesto actual en el historial.

Desde "Presupuestos guardados" se puede abrir un presupuesto anterior, cambiar su estado, eliminarlo, buscar por numero, cliente, direccion o notas, y filtrar por estado.

El resumen comercial muestra borradores, enviados, aceptados y rechazados con sus totales.

Sin sesion, el historial es local del navegador. Con sesion iniciada, los presupuestos se sincronizan automaticamente con la nube.

7. Dashboard

En "Dashboard" se ven metricas simples de los presupuestos guardados. Permite responder rapidamente:

  • Cuantos presupuestos se hicieron en el periodo.
  • Cuanto monto total se presupuesto.
  • Que porcentaje esta pendiente, aceptado o rechazado.
  • Que clientes concentran mas volumen.

El filtro de periodo permite revisar esta semana, este mes o los ultimos 3 meses. Los graficos muestran presupuestos por estado y principales clientes por volumen.

Si no hay presupuestos guardados, el dashboard queda vacio hasta guardar al menos uno.

8. Clientes

En "Clientes" se arma una base simple a partir de los presupuestos guardados. No hace falta cargar clientes a mano: Cotiza detecta el nombre del cliente desde cada presupuesto y muestra nombre, telefono, email, fecha de primer contacto, cantidad de presupuestos y volumen total.

Tambien se puede buscar por nombre o telefono, ordenar por volumen, fecha o nombre, y abrir un cliente para ver su historial completo de presupuestos.

Con sesion iniciada, al guardar un presupuesto Cotiza crea o vincula el cliente automaticamente en la nube, evitando duplicados.

9. Imprimir / Guardar PDF

El boton "Imprimir / Guardar PDF" abre la impresion del navegador. Desde ahi se puede imprimir o guardar como PDF.

La impresion incluye: datos del negocio, datos del cliente, fecha, partidas, notas, subtotal, impuesto, total y condiciones comerciales basicas.

La impresion no muestra: coste interno, margen interno ni estado comercial interno.

Revisar siempre cliente, numero, fecha, validez, partidas, cantidades, precios y total antes de enviarlo al cliente.

Casos practicos

Cotiza no sirve solo para pintar por m2. La misma logica se adapta a reparaciones, cambios de piezas, instalaciones, demoliciones y trabajos mixtos.

Reparacion puntual

Crear una plantilla simple llamada "Reparacion general" con: diagnostico/visita tecnica, hora oficial, material auxiliar y desplazamiento. Al crear el presupuesto usar cantidad = 1 y ajustar horas y materiales manualmente. Si la reparacion es muy incierta, agregar una linea manual como "Ajuste segun revision en obra".

Cambiar una pieza o instalar algo

Para trabajos repetibles como cambiar grifo, foco, enchufe o cerradura, usar cantidad como variable principal. Ejemplo: 4 enchufes → cantidad = 4. La plantilla calcula automaticamente piezas, material auxiliar, mano de obra y desplazamiento.

Ejemplo completo: cambiar un grifo

Primero revisar precios: grifo, material auxiliar, hora oficial y desplazamiento. Luego revisar el rendimiento "Horas cambio de grifo", por ejemplo 1,2 horas por cantidad.

La plantilla "Cambiar grifo" combina grifo por cantidad, material auxiliar por cantidad, hora oficial usando ese rendimiento y desplazamiento por trabajo.

Para crear el presupuesto real, entrar en Presupuesto, elegir "Cambiar grifo", poner cantidad = 1 y pulsar Crear presupuesto base. Si el cliente pone el grifo, borrar esa linea; si aparece un extra, agregarlo como linea manual.

Demolicion simple

Usar m2 si el esfuerzo depende del tamano. Plantilla sugerida: hora oficial por rendimiento m2, retirada de escombros por m2, saca/contenedor manual, proteccion de zona y desplazamiento. Si hay instalaciones, humedad o estructura, dejar una nota para revisar manualmente.

Trabajo por horas

Para trabajos sin medida clara: crear un precio "Hora oficial", una plantilla "Trabajo por horas" con cantidad directa y variable cantidad. Al presupuestar, usar cantidad = numero de horas estimadas. Sumar lineas manuales para materiales, parking o urgencia.

Presupuesto mixto

Muchos trabajos mezclan varias logicas. En esos casos, Cotiza calcula la parte repetible y la persona ajusta lo que depende de la visita. Usar lineas manuales para conceptos unicos, extras puntuales o cuando falta informacion.

Cuando usar lineas manuales

  • Concepto unico que no vale la pena convertir en plantilla.
  • Extra que aparece solo en ese trabajo.
  • Precio pactado de forma cerrada.
  • Falta informacion para calcularlo bien.
  • Aclaracion economica puntual.

Tutorial: crear un primer presupuesto

Este ejemplo cubre el flujo completo con un presupuesto de pintura de 25 m2.

  1. Configuracion. Ir a "Inicio" y cargar nombre del negocio, moneda EUR, impuesto 21, margen 20. Pulsar "Guardar configuracion".
  2. Precios. Ir a "Precios" y asegurarse de tener: pintura plastica interior, material auxiliar pintura, hora oficial y desplazamiento urbano.
  3. Consumos y tiempos. Verificar: litros pintura por m2, horas pintura por m2 y desplazamiento por trabajo.
  4. Presupuesto base. Ir a "Presupuesto", elegir plantilla "Pintar habitacion", superficie m2 = 25, cantidad = 1. Pulsar "Crear presupuesto base".
  5. Ajustar. Revisar cantidades y precios. Agregar si hace falta una linea extra, por ejemplo "Reparacion pequena de pared" a 35 EUR.
  6. Guardar. Cargar cliente, direccion, fecha, validez y estado "Borrador". Pulsar "Guardar presupuesto".
  7. Cambiar estado. Cuando se envie al cliente: Enviado. Si acepta: Aceptado. Si no avanza: Rechazado.
  8. Dashboard. Ir a "Dashboard" y comprobar totales, estados y top clientes.
  9. Clientes. Ir a "Clientes", buscar el cliente y abrir su detalle para ver el historial.
  10. PDF. Pulsar "Imprimir / Guardar PDF", revisar la vista y guardar desde el navegador.

Buenas practicas

  • Revisar siempre el presupuesto antes de enviarlo.
  • Mantener precios actualizados segun los costes reales.
  • Ajustar consumos y tiempos con experiencia real en obra.
  • Usar plantillas como base, no como verdad absoluta.
  • Revisar Dashboard y Clientes despues de guardar presupuestos.
  • Exportar backup si se cargan datos importantes.
  • No usar Cotiza como sistema fiscal, contable o legal.
  • Imprimir solo despues de revisar el resultado final.

Limites de la version actual

  • Sin login, los datos se guardan solo en el navegador. Con login se sincronizan en la nube.
  • Todavia no hay roles avanzados ni panel de administracion.
  • Clientes se detecta desde presupuestos; no es un CRM completo.
  • Dashboard muestra metricas basicas, no analitica avanzada.
  • No hay facturacion ni contabilidad.
  • No hay importador de Excel todavia.

Cotiza sirve para validar el flujo central: cargar datos propios, calcular una base editable y generar un presupuesto ordenado para el cliente.
