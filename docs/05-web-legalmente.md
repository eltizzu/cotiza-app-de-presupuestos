# Documento De Que Deberia Decir La Web Legalmente - Cotiza

## Mensajes que conviene incluir

La web deberia explicar que Cotiza:

- Es una herramienta de apoyo para crear presupuestos.
- Usa precios, rendimientos y reglas cargadas por el usuario.
- Permite editar manualmente el resultado final.
- No reemplaza criterio profesional, tecnico, legal o fiscal.
- En la demo actual, guarda datos localmente en el navegador.
- Si se usa backup JSON, el usuario debe custodiar el archivo exportado.

## Mensajes a evitar

Evitar afirmaciones absolutas como:

- "Presupuestos perfectos automaticamente".
- "Calcula siempre el precio correcto".
- "Cumple toda la normativa".
- "Garantiza rentabilidad".
- "Reemplaza a tu gestor o asesor".

## Textos recomendados

Texto breve:

> Cotiza genera propuestas de presupuesto a partir de los datos cargados por el usuario. Antes de enviar cualquier presupuesto, revisa precios, cantidades, impuestos, condiciones y alcance del trabajo.

Texto para pie legal:

> Cotiza es una herramienta de calculo y organizacion. El usuario es responsable de validar la informacion introducida, los resultados generados y el cumplimiento de sus obligaciones legales, fiscales y profesionales.

Texto para demo local:

> Esta demo guarda la informacion en el navegador. Si borras los datos del navegador o cambias de dispositivo, puedes perder la informacion salvo que exportes un backup.

Texto sobre backups:

> Los archivos de backup pueden contener informacion comercial y datos de clientes. Guarda esos archivos de forma segura.
