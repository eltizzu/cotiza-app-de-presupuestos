# Documento De Ecosistema / Ideas Futuras - Cotiza

## Ideas futuras ordenadas

### Importacion

- Importar listas de precios desde Excel o CSV.
- Importar catalogos de proveedores.
- Detectar cambios de precios.

### Inteligencia asistida

- Sugerir partidas faltantes.
- Ayudar a crear trabajos tipo desde texto.
- Revisar presupuestos en busca de omisiones.
- Explicar diferencias entre presupuestos.

La IA debe ser asistente, no autoridad.

### Gestion comercial

- Estados de presupuesto: borrador, enviado, aceptado, rechazado.
- Seguimiento de clientes.
- Recordatorios de seguimiento.
- Versiones de presupuesto.

### Operacion

- Agenda de trabajos.
- Control basico de obra.
- Partes de trabajo.
- Rentabilidad real vs estimada.

### Documentos

- Exportacion PDF avanzada.
- Plantillas visuales.
- Aceptacion online.
- Firma digital.

### Integraciones

- Excel y Google Sheets.
- Herramientas de facturacion.
- CRM simple.
- Proveedores.

## Criterio para sumar modulos

Un modulo futuro debe cumplir al menos una de estas condiciones:

- Ayuda a presupuestar mejor.
- Reduce trabajo repetitivo real.
- Evita errores frecuentes.
- Mejora claridad para el cliente.
- Aumenta control del margen.
